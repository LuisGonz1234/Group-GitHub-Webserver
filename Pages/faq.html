FAQ Page

  • Where is the visit count stored?

    • In the visits.txt file.

  • What module handles it?

    • counter.js

  • Where is the javascript running?

    • It is running on the server due to the code written in the server.js file.

  • What is a node webserver?

    • A javascript program that listens for a browser request, and sends back an answer. So in this project it is listening for the request on the port (get visitor count), it then reads the file visits.txt, and lastly it updates the count using the logic from counter.js and returns the answer back.
